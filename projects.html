Image Compression Using Huffman Algorithm

Huffman coding assigns a set of source symbols, taking into account the frequency and probability values of the symbols. While doing this, longer resource symbols are given to items with low probability/frequency, while short resource symbols are given to items with high frequency, increasing the efficiency of the code. It is very effective to use min heap to achieve this.Huffman coding can also be used for image compression operations. In this project, we saw that by using huffman coding, we can reduce the size of images without destroying the image. I performed compression operations on gray pictures and colored pictures by calculating both with the array I produced from the image and with the difference array I produced by performing some operations on the array. When I decompressed each stage, I saw that I could reach the original image.

Spine Posture Detection Using Machine Learning

In this innovative project, my friend and I collaborated to develop a system aimed at detecting bad spine posture utilizing advanced machine learning algorithms. The primary objective was to create a robust and accurate system capable of identifying improper spine alignment, contributing to better posture awareness and potentially preventing associated health issues. The developed system holds significant potential in promoting spinal health awareness, especially in contexts where individuals may not have immediate access to healthcare professionals. By accurately detecting bad spine posture, the system can provide timely feedback to users, encouraging them to correct their posture and adopt healthier habits.
